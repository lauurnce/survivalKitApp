# Dark Mode Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a fixed top-right light/dark toggle that persists in `localStorage` and defaults to light mode.

**Architecture:** Enable Tailwind's `darkMode: 'class'` strategy, remap all CSS custom properties inside a `.dark {}` block in `globals.css`, and add a `ThemeToggle` client component rendered in `layout.tsx`. A tiny inline script in `<head>` prevents flash-of-wrong-theme for returning dark-mode users by synchronously applying the `dark` class before paint.

**Tech Stack:** Next.js App Router, Tailwind CSS, React (client component), inline SVG icons, `localStorage`

## Global Constraints

- No new npm dependencies — inline SVG only for icons
- Always-light default: do not read `prefers-color-scheme`
- `localStorage` key is `'theme'`, values are `'light'` or `'dark'`
- `dark` class lives on `document.documentElement` (`<html>`)
- No per-component `dark:` Tailwind class changes — CSS variables handle everything

### Task 2: Create ThemeToggle component

**Files:**
- Create: `components/ThemeToggle.tsx`

**Interfaces:**
- Consumes: `localStorage` key `'theme'` (values `'light'` | `'dark'`); `document.documentElement.classList`
- Produces: `<ThemeToggle />` — a self-contained fixed-position button, no props required

- [ ] **Step 1: Create `components/ThemeToggle.tsx`**

```tsx
"use client";

import { useEffect, useState } from "react";

export function ThemeToggle() {
  const [theme, setTheme] = useState<"light" | "dark">("light");

  useEffect(() => {
    const stored = localStorage.getItem("theme");
    if (stored === "dark") {
      setTheme("dark");
      document.documentElement.classList.add("dark");
    } else {
      setTheme("light");
      document.documentElement.classList.remove("dark");
    }
  }, []);

  function toggle() {
    const next = theme === "light" ? "dark" : "light";
    setTheme(next);
    localStorage.setItem("theme", next);
    if (next === "dark") {
      document.documentElement.classList.add("dark");
    } else {
      document.documentElement.classList.remove("dark");
    }
  }

  return (
    <button
      onClick={toggle}
      suppressHydrationWarning
      aria-label={theme === "dark" ? "Switch to light mode" : "Switch to dark mode"}
      className="fixed top-4 right-4 z-50 w-9 h-9 flex items-center justify-center border border-ink-faint/30 bg-paper text-ink hover:bg-ink hover:text-paper transition-colors duration-150"
    >
      {theme === "dark" ? (
        // Sun icon
        <svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" strokeWidth="2" strokeLinecap="round" strokeLinejoin="round">
          <circle cx="12" cy="12" r="5"/>
          <line x1="12" y1="1" x2="12" y2="3"/>
          <line x1="12" y1="21" x2="12" y2="23"/>
          <line x1="4.22" y1="4.22" x2="5.64" y2="5.64"/>
          <line x1="18.36" y1="18.36" x2="19.78" y2="19.78"/>
          <line x1="1" y1="12" x2="3" y2="12"/>
          <line x1="21" y1="12" x2="23" y2="12"/>
          <line x1="4.22" y1="19.78" x2="5.64" y2="18.36"/>
          <line x1="18.36" y1="5.64" x2="19.78" y2="4.22"/>
        </svg>
      ) : (
        // Moon icon
        <svg xmlns="http://www.w3.org/2000/svg" width="16" height="16" viewBox="0 0 24 24" fill="none" stroke="currentColor" strokeWidth="2" strokeLinecap="round" strokeLinejoin="round">
          <path d="M21 12.79A9 9 0 1 1 11.21 3 7 7 0 0 0 21 12.79z"/>
        </svg>
      )}
    </button>
  );
}
```
